## Usage

1. Clone the repository
2. Open index.html in your browser
3. Customize content and styling as needed

## Customization

The site uses CSS variables for easy customization:

```css
:root {
  --bg-primary: #ffffff;
  --bg-secondary: #f8f9fa;
  --text-primary: #212529;
  --text-secondary: #6c757d;
  --border-color: #dee2e6;
  --teal: #20c997;
}
```
